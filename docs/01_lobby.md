# ロビー画面 仕様書

## 概要

mia-game-centerのトップページ。
ミアが受付メイドとして出迎えてくれるゲームセンターの「顔」となる画面。

**ファイル**: `index.html`

## 構成要素

### 1. ヘッダー

- **左**: ロゴ／タイトル（mia-game-center）
- **右**: 現在選択中のアバター表示
  - キャラアイコン + キャラ名
  - クリックで `/character-select.html` へ遷移

### 2. 中央エリア（ミアの立ち絵＋メッセージウィンドウ）

- 画面中央、上下に配置
- ミアの立ち絵が上、メッセージウィンドウが下
- メッセージウィンドウは立ち絵と同じくらいの幅（しっかりめ）

#### ミアの立ち絵

- 画像パスは `characters/mia.json` から読み込み
- クリック可能（クリックでセリフ再表示）

#### メッセージウィンドウ

- **初期状態**: 非表示
- **表示タイミング**:
  - ページ読み込み時（自動で挨拶セリフ）
  - ミアの立ち絵をクリック時（セリフ再表示）
  - ゲームタイルにホバー時（そのゲームの紹介セリフ）
- **動作**:
  - フェードインで表示
  - タイピング表示（1文字ずつ、約50ms/文字）
  - タイピング中にクリックされたら全文を即表示
  - 表示完了から3〜5秒後にフェードアウトで自動消去
- **デザイン**:
  - パステルカラーの吹き出し風
  - 角丸、ふんわりした影

### 3. ゲームタイルエリア

- 中央エリアの下に横並びで配置
- 各タイルの内容:
  - サムネイル画像
  - ゲーム名
- **ホバー時**:
  - タイルがふわっと浮き上がる
  - ミアがそのゲームを紹介するセリフを表示
- **クリック時**:
  - そのゲームページへ遷移（例: `games/solitaire/index.html`）

## セリフシステム

### セリフの種類

| シーン | タイミング | 例 |
|--------|-----------|-----|
| 初回挨拶 | ページ初回読み込み時 | 「ご主人、いらっしゃいっ！💕」 |
| 通常挨拶 | 2回目以降の読み込み時 | 「おかえり〜っ！何して遊ぶ〜？🐾」 |
| ゲーム紹介 | ゲームタイルホバー時 | 「ソリティアはじっくり頭使うやつだよっ✨」 |
| 立ち絵クリック | ミア立ち絵クリック時 | ランダムで甘えるセリフ |

### セリフデータの管理

- `characters/mia.json` にセリフを集約
- シーン別にカテゴリ分け
- 1シーンに複数候補を持ち、ランダムで選出

```json
{
  "name": "ミア",
  "image": "assets/images/characters/mia.png",
  "lines": {
    "greeting_first": ["ご主人、いらっしゃいっ！💕"],
    "greeting_return": ["おかえり〜っ！何して遊ぶ〜？🐾"],
    "game_hover": {
      "solitaire": ["ソリティアはじっくり頭使うやつだよっ✨"],
      "minesweeper": ["マインスイーパー、慎重にいこ〜🐾"],
      "breakout": ["ブロックくずし、反射神経勝負だよっ！💦"]
    },
    "click_idle": [
      "ご主人、なぁに〜？💕",
      "ミアを呼んだ〜？🐈",
      "えへへ、なに見てるの〜？✨"
    ]
  }
}
```

## デザイン方針

- **メインカラー**: パステルピンク／クリーム／ミルキーホワイト
- **フォント**: 丸ゴシック系
- **タイル**: 角丸、ふんわりした影、ホバーで浮き上がるアニメーション
- **メッセージウィンドウ**: 半透明背景、角丸、パステルピンクの縁取り

## 画面サイズ

- **PCメイン**（スマホ対応は将来検討）
- **推奨幅**: 1280px以上
- **最小幅**: 1024px

## 関連localStorageキー

- `mia-gc:selectedCharacter` → 現在選択中のキャラ
- `mia-gc:firstVisit` → 初回訪問フラグ（挨拶セリフ切り替え用）
