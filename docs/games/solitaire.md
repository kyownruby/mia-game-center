# ソリティア 仕様書

## 概要

mia-game-center内のミニゲーム第1弾。
クラシックなクロンダイク・ソリティアを、パステルカラーで可愛くアレンジ。

**ゲームページ**: `games/solitaire/index.html`

## ゲームルール

### 基本ルール（クロンダイク）

- **使用デッキ**: トランプ1組（52枚、ジョーカーなし）
- **目的**: すべてのカードを組札（ファウンデーション）に、各スート別にA→Kの順で積み上げる

### 初期配置

- **場札（タブロー）**: 7列に配置
  - 1列目に1枚、2列目に2枚 …… 7列目に7枚
  - 各列の一番上のカードだけ表向き、それ以外は裏向き
- **山札（ストック）**: 残り24枚を裏向きで配置
- **めくり札（ウェイスト）**: 空の状態でスタート
- **組札（ファウンデーション）**: 4ヶ所（♠♥♦♣）、すべて空でスタート

### カード移動ルール

- **場札 → 場札**: 数字が1つ小さく、色が異なるカード上に置ける（赤→黒、黒→赤）
- **場札 → 組札**: 同じスートで、組札の一番上のカードより1つ大きい数字（Aから順番）
- **山札 → 場札／組札**: 上記のルールに従って移動可能
- **空の場札にはKのみ**置ける
- **裏向きカードの上に重ねられているカードを移動した場合、裏向きカードは自動で表向きになる**
- **連続したカード列の移動**: 場札内で、ルールに従って並んでいる複数枚をまとめて移動可能

### 山札のめくり方（2モード）

- **Draw 1**: 山札から1枚ずつめくる（易しい）
- **Draw 3**: 山札から3枚ずつめくる（難しい）
- **新規ゲーム開始時にモーダルで選択**

### クリア条件

- 4つの組札がすべてK（13枚×4スート＝52枚）で埋まる

## 操作方法

すべての操作方法を実装する：

| 操作 | 動作 |
|------|------|
| ドラッグ&ドロップ | カードを目的の場所へ移動 |
| クリック2回 | 1回目: 移動元のカード選択／2回目: 移動先を指定して移動 |
| ダブルクリック | カードを組札へ自動移動（移動可能な場合） |
| 右クリック | ヒント表示（次の一手を提案） |
| 山札クリック | カードをめくる（Draw 1 or Draw 3） |
| 山札が空のときクリック | めくり札を山札に戻す |

## スコアシステム（Windows準拠・タイマー減点なし）

| アクション | スコア |
|-----------|--------|
| 山札 → 場札にカード移動 | +10点 |
| 場札 → 組札にカード移動 | +10点 |
| 山札 → 組札に直接移動 | +5点 |
| 場札の裏向きカードを表向きにする | +5点 |
| 組札 → 場札に戻す | -15点 |
| クリア時ボーナス | 残り時間に応じて加点（タイマー減点はなし） |

※ プレイ時間は計測・表示するが、スコアからの減点はしない

## 機能仕様

### 1. 新規ゲーム

- ヘッダーの「新規ゲーム」ボタン
- クリックで**モーダル表示**：「Draw 1」「Draw 3」を選択
- 選択後、盤面リセットして開始

### 2. Undo（一手戻る）

- ヘッダーの「Undo」ボタン
- **何手でも戻れる**
- **回数制限・ペナルティなし**
- 直前の状態（カード配置・スコア）に完全復元

### 3. ヒント

- ヘッダーの「ヒント」ボタン
- **次の一手を1つ提案**：移動可能なカードと移動先をハイライト表示
- **ペナルティなし**
- 移動可能な手がない場合は「うーん、動かせるカードないみたい…💦」とミアがコメント

### 4. 自動完成（オートコンプリート）

- **発動条件**: 場札のすべてのカードが表向きになった時点で「自動完成」ボタン出現
- クリックで一気にカードが組札へ移動するアニメーション
- スコア計算は通常通り

### 5. タイマー

- ゲーム開始から経過時間を表示（mm:ss形式）
- ヘッダー右側に常時表示
- スコアには影響しない（表示のみ）

## ハイスコア保存

### localStorageキー設計

```
mia-gc:scores:solitaire:draw1 → {best, plays, fastestTime, totalWins, lastPlayed}
mia-gc:scores:solitaire:draw3 → {best, plays, fastestTime, totalWins, lastPlayed}
```

### 保存内容

- **best**: 最高スコア
- **plays**: プレイ回数（クリアしていなくてもカウント）
- **fastestTime**: 最速クリアタイム（秒）
- **totalWins**: クリア回数
- **lastPlayed**: 最終プレイ日時（ISO文字列）

### 表示

- ゲーム終了時のリザルト画面で表示
- ロビーのタイル ホバー時にもベストスコアを表示（任意）

### 配置

- **ヘッダー**: 上部固定。新規ゲーム／Undo／ヒント／ロビーへ戻る／スコア／タイマー
- **盤面**: 中央。組札（右上）、山札＋めくり札（左上）、場札7列（中央）
- **キャラエリア**: 右下に固定。選択キャラ（ミア or ご主人）の立ち絵＋セリフウィンドウ

### キャラクター表示

- `localStorage`から`mia-gc:selectedCharacter`を読み込み、対応するキャラの立ち絵を表示
- キャラデータは`characters/{キャラID}.json`から取得
- セリフウィンドウはロビーと同じ仕様（タイピング表示、フェードイン・アウト）

## キャラセリフ仕様

選択キャラ（ミア or ご主人）のセリフを各シーンで表示。
セリフは各キャラのJSONファイルに集約。

### セリフ発火タイミング

| シーン | タイミング |
|--------|-----------|
| ゲーム開始 | 盤面表示直後 |
| 組札への移動成功 | カードが組札に置かれた時 |
| Undo実行 | Undoボタン押下時 |
| 長時間無操作 | 30秒以上カード移動がない時 |
| ヒント使用 | ヒントボタン押下時 |
| 自動完成発動 | 自動完成ボタン押下時 |
| クリア | 全カードが組札に揃った時 |
| 行き詰まり | 移動可能な手がない時 |

## デザイン方針

### カード

- **シンプルなトランプデザイン**
- 表：白背景、数字とスートマークを中央＋角に表示
- 裏：パステルピンクのパターン柄（mia-game-centerオリジナル）
- 角丸、ふんわりした影

### 盤面

- **パステルカラーの背景**（ミルキーホワイト or 淡いピンク）
- 各エリア（山札・組札・場札）の枠は淡いボーダー
- 全体的に柔らかい印象

### ボタン

- 角丸、ぷにっとした質感
- パステルピンクを基調

- `shared/characters.js`: キャラデータ読み込み
