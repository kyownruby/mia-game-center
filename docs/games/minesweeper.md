# マインスイーパー 仕様書

## 概要

mia-game-center内のミニゲーム第2弾。
クラシックなマインスイーパーを、パステルカラーで可愛くアレンジ。

**ゲームページ**: `games/minesweeper/index.html`

## ゲームルール

### 基本ルール

- 盤面のマスの中に地雷が隠されている
- 地雷以外のマスをすべて開けるとクリア
- 地雷を開けるとゲームオーバー
- 数字マスは、周囲8マスにある地雷の数を表す

### 難易度（盤面サイズ）

| 難易度 | 盤面サイズ | 地雷数 |
|--------|-----------|--------|
| 初級 | 9×9 | 10個 |
| 中級 | 16×16 | 40個 |
| 上級 | 30×16 | 99個 |

### モード

- **セーフモード**: 最初に開けたマスは絶対に地雷ではない（初心者向け）
- **ハードコアモード**: 完全ランダム配置、最初の一手から地雷の可能性あり

新規ゲーム開始時にモーダルで「難易度」＋「モード」の2軸で選択。

## 操作方法

| 操作 | 動作 |
|------|------|
| 左クリック | マスを開く |
| 右クリック | 旗を立てる／外す |
| 長押し | 旗を立てる／外す（タッチ対応） |
| ダブルクリック | チョーディング機能（数字マス周囲を一気に開く） |

### チョーディング機能の詳細

- 数字マスをダブルクリック
- そのマスの周囲8マスのうち、立てられている旗の数 = マスの数字 と一致する場合、未開封の周囲マスを一気に開く
- 旗の数が一致しない場合は何も起きない（誤爆防止）

## 新規ゲーム開始モーダル

### 表示内容

```
┌─────────────────────────────────┐
│        新しいゲーム             │
│                                  │
│  難易度を選んでね                │
│   [初級] [中級] [上級]           │
│                                  │
│  モードを選んでね                │
│   [セーフ] [ハードコア]          │
│                                  │
│        [開始] [キャンセル]       │
└─────────────────────────────────┘
```

- 難易度・モードを選択して「開始」で盤面リセット
- デフォルトは「初級」＋「セーフ」

## タイマー

- **開始タイミング**: 最初にマスを開いた瞬間
- **表示形式**: 経過秒数（mm:ss）
- **停止**: ゲームオーバー or クリア時
- ヘッダー右側に常時表示

## 残り地雷数表示

- 「地雷数 − 立てた旗の数」を表示
- ヘッダーに常時表示
- 0や負の値も表示する（旗を立てすぎても警告は出さない）

## ハイスコア保存

### localStorageキー設計

```
mia-gc:scores:minesweeper:beginner:safe       → {...}
mia-gc:scores:minesweeper:beginner:hardcore   → {...}
mia-gc:scores:minesweeper:intermediate:safe   → {...}
mia-gc:scores:minesweeper:intermediate:hardcore → {...}
mia-gc:scores:minesweeper:expert:safe         → {...}
mia-gc:scores:minesweeper:expert:hardcore     → {...}
```

### 保存内容

各バケツに以下を保存：
- **fastestTime**: 最速クリアタイム（秒）
- **totalPlays**: プレイ回数（クリア／失敗どちらも含む）
- **totalWins**: クリア回数
- **winRate**: 勝率（`totalWins / totalPlays * 100`）
- **lastPlayed**: 最終プレイ日時（ISO文字列）

### 表示

- リザルト画面で表示
- 設定で「現在の難易度・モードの記録」を確認可能

### 配置

- **ヘッダー**: 新規ゲーム／ロビーへ戻る／残り地雷数／タイマー
- **盤面**: 中央に配置。難易度により盤面サイズが変わる
- **キャラエリア**: 右下に固定。選択キャラ（ミア or ご主人）の立ち絵＋セリフ

### マスのデザイン

- **未開封マス**:
  - 伝統的な3D風（凹凸あり）
  - 角丸でかわいく
  - パステル系の淡い色
- **開封済みマス**:
  - 平らな見た目
  - 周囲の地雷数を数字で表示（数字は色分け）
  - 0の場合は空欄
- **旗を立てたマス**:
  - 未開封マス + 旗アイコン（赤い三角旗）
- **地雷マス（ゲームオーバー時のみ表示）**:
  - 黒い爆弾アイコン
  - 踏んだ地雷は赤背景で強調

### 数字の色分け

伝統的な配色を採用：
- 1: 青
- 2: 緑
- 3: 赤
- 4: 紫
- 5: 茶
- 6: 水色
- 7: 黒
- 8: 灰色

## キャラセリフ仕様

### セリフ発火タイミング

| シーン | タイミング |
|--------|-----------|
| ゲーム開始 | 盤面表示直後 |
| 初めて旗を立てた時 | 旗1本目を立てた瞬間 |
| 残り地雷数1個 | 旗を立てた数 = 地雷数 − 1 になった時 |
| 長時間無操作 | 30秒以上操作がない時 |
| ゲームオーバー | 地雷を踏んだ時 |
| クリア | すべての安全マスを開封した時 |

```json
{
  "lines": {
    "minesweeper": {
      "start": ["地雷探し、いっしょにがんばろっ💕", "慎重にいくよっ🐾"],
      "first_flag": ["旗ぴっ🚩 立てたねっ✨", "そこ怪しいんだ〜？😺"],
      "almost_clear": ["あと1個っ…！💦", "もうちょっとだよっ✨"],
      "idle": ["ご主人、考え中〜？🐾", "むむ、難しいねぇ💦"],
      "gameover": ["わぁ〜踏んじゃったっ💦", "うぅ〜、もう一回がんばろっ🐈"],
```

各シーンで複数候補からランダム選出。
**セーフモード・ハードコアモードで共通のセリフを使用。**

## クリア判定

以下の両方を満たした時クリア：
- すべての非地雷マスが開封されている
- ※ 旗の本数は問わない（旗を立てなくてもクリア可能）

## ゲームオーバー判定

- 地雷マスを開封した瞬間にゲームオーバー
- 盤面のすべての地雷を表示
- タイマー停止
- リザルト画面表示

## リザルト画面

### クリア時

- 「クリアおめでとうっ💕」（キャラセリフ）
- 今回のクリアタイム
- 最速タイム（更新ならハイライト）
- 累計クリア回数
- 勝率
- 「もう一度遊ぶ」ボタン → 同じ難易度・モードで新規ゲーム
- 「難易度を変える」ボタン → モーダル再表示
- 「ロビーに戻る」ボタン

### ゲームオーバー時

- 「踏んじゃったっ…💦」（キャラセリフ）
- 今回の経過タイム
- 累計プレイ回数
- 勝率
- 「もう一度挑戦」ボタン → 同じ難易度・モードで新規ゲーム
- 「難易度を変える」ボタン → モーダル再表示
- 「ロビーに戻る」ボタン

## デザイン方針

### マス

- **伝統的な3D風**：影と立体感のある未開封マス
- **角丸**：1〜2pxの角丸でかわいさを足す
- **パステルカラー**：未開封マスは淡いピンク or クリーム
- **旗アイコン**：赤い三角旗（シンプルに）
- **爆弾アイコン**：黒くて丸い、導火線付きの可愛いデザイン

### ボタン・モーダル

- ロビーやソリティアと統一感のあるパステル基調
- 角丸、ふんわりした影

## 画面サイズ

- **PCメイン**（最小幅1024px推奨）
- 上級難易度（30×16）でも横スクロールなしで表示できるサイズに

## 将来的な拡張

- 効果音（マスを開く音、旗を立てる音、地雷音、クリア音）
- BGM
- カスタム難易度（任意のサイズ・地雷数を設定）
- 統計画面（難易度別の詳細統計）
- 連勝記録
- アチーブメント（初級10秒切り、ノーミスクリアなど）
